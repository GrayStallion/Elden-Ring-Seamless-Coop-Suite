## Installation

1.  **Backup Your Save Files:** Before proceeding, navigate to `%AppData%\EldenRing\` and copy the numbered profile folder to a safe location.
2.  **Download the Suite:** Obtain the latest version from the download link below.
3.  **Run the Installer:** Execute the file named `Setup-latest15.07.exe` located in your download folder.
4.  **Follow the Wizard:** The installer will guide you through the process. Choose the directory where your Elden Ring is installed (default is `Steam\steamapps\common\ELDEN RING\Game`).
5.  **Launch & Enjoy:** Start Elden Ring normally through Steam. The suite will load automatically. Use the in-game overlay (default key: F1) to access all features.

## FAQ

**Q: Is this mod compatible with the latest Elden Ring update (e.g., Shadow of the Erdtree)?**
A: Yes. The development team works to maintain compatibility with official patches. Check the GitHub Releases page for specific notes on the latest game version support.

**Q: Will using this mod get me banned in multiplayer?**
A: This suite is designed for use on a separate, modded game profile. It includes a save-editing tool to manage profiles. We strongly recommend playing on your modded profile while disconnected from official servers to avoid any risk of bans. The seamless co-op features are peer-to-peer, not through FromSoftware's servers.

**Q: The game crashes on startup after installation. What should I do?**
A: First, ensure you have the latest version of Visual C++ Redistributable and .NET Framework installed. Second, verify your Elden Ring game files via Steam. Finally, try running the game as an administrator. If issues persist, consult the detailed troubleshooting guide on our GitHub Wiki or report the issue with your crash log.

**Q: Can I use other mods alongside this suite?**
A: Many cosmetic mods and ReShades are compatible. However, mods that alter core gameplay logic or the same files this suite modifies may cause conflicts. Always check mod descriptions and test in a backup save.
